🧠 Perceptron Lab

A simple, interactive way to see how a perceptron actually learns — right in your browser.

🌐 Live Demo:
Perceptron Lab — Basam Babar

👀 What is this?

I built Perceptron Lab to make one of the most important machine learning concepts easier to understand.

Instead of just reading formulas, you can:

train a perceptron
tweak parameters
and watch how the decision boundary changes in real time

It’s meant to be a hands-on way to explore how basic neural networks work.

⚡ What you can do
🎯 Train a perceptron and watch it learn step by step
📊 See accuracy improve over time
⚙️ Adjust:
learning rate
number of epochs
🔄 Switch between datasets:
OR
AND
NAND
NOR
📈 Visualize the decision boundary
📋 Compare predictions with actual outputs in a live table
🧠 The idea behind it

At its core, the perceptron is just trying to separate data using a line.

Each step:

it makes a prediction
checks if it’s wrong
adjusts weights slightly

That’s it — but seeing it happen visually makes everything click.

🛠️ Built with
HTML
CSS
Vanilla JavaScript
Chart.js (for the accuracy graph)

No frameworks — everything is built from scratch.
